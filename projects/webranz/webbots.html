What are Web bots?

Web bots are programs that simulate user browsing behavior to automatically interact with web sites. Web bots read the HTML code, analyze the content and interact with the web applications. The common objectives of web bots are searching, scraping and impersonation. Web bots are widely used to automate various tasks such as gathering information on web sites.

What they can do?

Web bots can do pretty much everything that human can do such as gathering information, posting articles, downloading files, purchasing goods, and etc. In addition, since they are bots, they can do a number of tasks quickly and easily. Just like humans, web bots identifies elements in the web sites (e.g., download button), and trigger events (e.g., onclick event) to get the desirable results.

Why they are bad?

While web bots are useful in practice, there are also malicious web bots. Such malicious web bots can be used to launch DoS attacks. For example, if there are too many bots connected to a web site, real users may not be able to get appropriate services.
Also, even the benign bots can affect other users because a real user may not get serviced if there are many bots hogging the server's resources.

Web bots in our experiments

In our experiments, we have few interesting web bots. Just like people need to find a certain element to achieve a goal, web bots also need to identify DOM elements in the page. However, while people use their eye-sight to understand the interface, web bots use attributes of DOM elements such as ids and names. It hierarchically searches for some certain DOM elements using ids or names. As WebRanz randomizes such ids and names of DOM elements, it naturally nullifies the web bots' implementations.

Followings are some of the web bots we have tested and brief descriptions of them.
1. Lenovo outlet: repeatedly crawling web sites until it gets certain response (a deal in the store). [Consuming server resources and preventing other users from getting deals]
2. YelpCrawl: crawling a number of restaurant data from yelp website. [Hogging server resources, degrading the quality of service]
3. Amazon Review Downloader: crawling a number of reviewers from amazon. [Hogging server resources, degrading the quality of service]
4. Groupon Crawler: crawling all deals in America. [Hogging server resources, degrading the quality of service]
