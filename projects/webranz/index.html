What is WebRanz?

WebRanz is a tool that randomizes web pages so that the contents of the randomized web pages can be protected and preserved from Ad-blockers[*] and Web bots[+]. For example, the advertisements in the randomized web pages are not removed by Adblockers such as AdBlock Plus. In addition, it naturally protects the web pages from web bot attacks, as most web bots rely on predefined id and classnames to locate specific DOM elements. For example, Figure 1 shows a running example. It shows three screen shots of a web-site without Adblock Plus and with Adblock Plus, as well as randomized website with Adblock Plus. Our high-level goal is to preserve all the visible elements and their layouts including advertisements against Adblock Plus by randomizing the web pages. The red boxes represent the advertisements and their places. The red arrows point the Adblock Plus icons. The gray icon means it is disabled, the red icon means it is enabled and preventing advertisements.


Figure 1. Screen shots side by side.

As shown in Figure 1, the first case which is the original web page without Adblock Plus shows two advertisements. In the second case, the original web page with Adblock Plus, there is no advertisements showing as they are blocked by Adblock Plus. In the last case, the randomized web page with Adblock Plus, all advertisements are shown even with Adblock Plus due to our randomizations.

[*] What is Ad-blocker?

Ad-blocker is a piece of software that allows a user to roam the web without encountering any Ads. In particular, it utilizes network control and in-page manipulation to help users block most online advertisements. Network control barricades HTTP requests to Ads and thus prevents themfrom loading. In-page manipulation looks up Ads based on pre-determined patterns and makes them invisible. Want to know more about Ad-blockers? Please check "Ad-blockers?" page out.

[+] What are Web bots?

Web bots are programs that simulate user browsing behavior. They read the HTML code, analyze the content and interact with the web app just like humans. Web bots are commonly used for various purposes such as searching, scraping and impersonation. According to a recent study on 20,000 popular websites by Incapsula Inc., out of 15 billion visits observed, 56% of all traffic is generated by bots. Want to know more about Web bots? Please check "Web bots?" page out.

Why WebRanz?

WebRanz protects and preserve the contents including legitimate advertisements from Adblockers or web bots.
While adblockers have many benefits to the end users, it is also destroying web service ecosystems especially those who depend on the advertisement incomes. WebRanz protects such web services and makes the web ecosystem sustainable.
In addition, WebRanz also prevents web bots so that the service quality is not affected by hogging web bots.
More details can be found in "Why WebRanz?" page.

How it works?

WebRanz analyzes web pages and randomizes id and class names as well as CSS contents as any of these names can be used to locate or identify a specific element on a web page. Figure 2 shows an example of such randomization on HTML code. On the left side, the code of original web page is shown. On the right side, the randomized web page is shown. The red boxes are ids and classnames. On the left side, the first red box contains "header", "logo", and "tabs", which are transformed into "rand_id_29099", "rand_class_892", and "rand_class_27487" respectively.


Figure 2. An example source code of a randomized web page.

By randomizing the ids, classnames, and urls, it can protect web contents from other plug-ins which try to locate or identify certain elements in web pages. However, benign accesses to DOM objects can be also affected. To handle that, WebRanz dynamically maintains a mapping between randomized ids/classnames and original ids/classnames and translates them at runtime.
