What are Ad-blockers?

Ad-blockering or ad-filtering is removing or altering advertisements in a web page, and Ad-blockers are essentially programs that implement ad-blocking methods.

While there are many different ways to implement Ad-blockers, the most common and popular ad-blockers are browser extensions.
For instance, Adblock Plus is the leading browser extension for blocking ads. A user can install the extension on the Chrome web-browser in the Web Store. Once it is installed, it automatically detects and prevents advertisements in a web page. Specifically, it removes the advertisements by blocking the advertisement loading process via URL filtering and hiding the DOM elements that have certain ids or names.

How it works?

Most ad-blockers prevent advertisements via URL and DOM elements filters. The URL filter is a list of URLs that are used to deliver advertisements. Once such URL is detected, Ad-blockers block network connections to and from such URLs, preventing the delivery of advertisements.
However, some advertisements are delivered through URLs that are also used to deliver non-ad contents. As these URLs are indistinguishable, preventing such URLs often breaks the website. To block such advertisements cannot be blocked via URL filters, Ad-blockers identify DOM elements related to advertisements on the web page and hide the elements.
To implement URL and DOM element filters, Ad-blockers often use a predefined list that includes all known advertisements' element ids, names, and URLs.

In particular, EasyList is one of the most popular list. Figure 4 shows some parts of the EasyList.
